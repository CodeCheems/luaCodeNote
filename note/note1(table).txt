    表（ Table ）是 lua 语言中最主要（事实上 是唯一的）和强大的数据结构 使用表， Lua
语言可以 一种简单、统一且高效的方式表示数组、集合、记录和其他很多数据结构 Lua
语言也使用表来表示包（ package ）和其他对 当调用函数 math.sin ，我们可能认为是
“调用了 math 库中函数 sin ；而对 Lua 语言来说，其实际含义是“以字符串 sin 为键检
索表 math ”。
Lua 言中的表本质上是一种辅助数组（ associative array ），这种数组不仅可以使用数值
作为索引，由可以使用字符串或其他任意类型的值作为索引（ nil 除外）
Lua 言中的表要么是值要么是变量 ，它们都是对象 bject 如果读者对 Java
Scheme 中的数组 比较熟悉 ，那么应该很容易理解上述概念 可以认为，表是一种动态分配
的对象，程序只能操作指向表的引用（或指针） 除此以外，lua 语言不会进行隐藏的拷贝
( hidden copies ）或创建新的表
